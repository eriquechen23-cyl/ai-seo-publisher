WordPress Theme Styling Milestone

2026-07-03 21:37 Asia/Taipei · ai-seo-publisher · Codex

本階段新增本地 WordPress preview theme，改善前台首頁與文章頁的展示品質。實作只觸及 wp-content/themes theme layer，不修改 WordPress core。

架構流程

實作摘要

Theme

新增 AI SEO Demo theme，包含首頁、文章頁、header/footer 與 CSS。

Script

start-wordpress.ps1 會複製 theme 並用 WP-CLI 啟用。

Presentation

文章頁現在有品牌 header、內容容器、側欄說明、列表與 CTA 樣式。

Boundary

只調整 theme layer；不修改 core、不改 REST API、不污染文章 HTML。

驗證狀態

  • WP-CLI 回報 AI SEO Demo theme 已 Active。
  • php -l 檢查 functions.php、index.php、single.php、header.php、footer.php 無語法錯誤。
  • http://localhost:8080 回 200，HTML 載入 ai-seo-demo-style-css。
  • style.css URL 回 200。
