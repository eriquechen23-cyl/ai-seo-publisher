WordPress Modern Theme Refinement

2026-07-03 22:00 Asia/Taipei · ai-seo-publisher · Codex

根據使用者截圖回饋，第一版 theme 仍像傳統卡片頁且標題過大。本階段重做 single post 文章頁，使其更接近現代內容產品頁。

設計修正

Hero

改為全寬深色 hero，主標題控制在合理閱讀尺度。

Content

正文區變成 760px 閱讀欄，並隱藏 LLM 內容內重複的第一個 h1。

Sidebar

側欄改為輕量 metadata，不再使用突兀卡片。

Responsive

手機寬度 390px 無水平溢出，標題降為 34px。

驗證

  • WP-CLI 確認 AI SEO Demo theme active，版本為 1.1.0。
  • php -l 檢查 single.php 與 functions.php 無語法錯誤。
  • style.css URL 回 200，且內容包含 Version: 1.1.0。
  • 瀏覽器桌面檢查：內容內第一個 h1 已隱藏，正文寬度為 760px。
  • 瀏覽器手機檢查：390px viewport 無水平溢出。
